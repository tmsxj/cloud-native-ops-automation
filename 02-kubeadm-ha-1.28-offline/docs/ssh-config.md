## 带宽情况

| 链路 | 带宽 | 传输速度 |
|:-----|:-----|:---------|
| us → tx | 30M | ~3MB/s |
| tx → hb | 1M | ~300KB/s |
| us → hb | 不稳定 | 30KB-3MB/s (闲时3MB, 忙时30KB) |

## 镜像传输策略

### 推荐策略：us → tx → hb（中转）

```
us拉取镜像 → tx临时存储 → hb Harbor仓库
   3MB/s      300KB/s      存储
```

优点：
- us到tx稳定3MB/s
- tx作为中转缓冲
- 适合大文件传输

### 备选策略：us → hb 直传

适用于：
- tx磁盘空间不足
- 镜像较小（<100MB）
- 闲时传输（3MB/s）

### 镜像同步命令

```bash
# 推荐：经过tx中转（稳定）
./scripts/sync_via_tx.sh -f images.txt

# 备选：us直传hb（闲时使用）
./scripts/sync_direct.sh -f images.txt

# 同步单个镜像
./scripts/sync_via_tx.sh nginx:latest 300
```

## SSH免密配置

### 在tx上配置密钥

```bash
# 生成密钥
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519 -N ""

# 复制到所有节点
ssh-copy-id -i ~/.ssh/id_ed25519.pub m1
ssh-copy-id -i ~/.ssh/id_ed25519.pub m2
ssh-copy-id -i ~/.ssh/id_ed25519.pub m3
ssh-copy-id -i ~/.ssh/id_ed25519.pub w1
ssh-copy-id -i ~/.ssh/id_ed25519.pub w2
ssh-copy-id -i ~/.ssh/id_ed25519.pub hb

# 配置SSH config（见上方配置）
```

### 验证连接

```bash
# 测试所有节点连接
ssh m1 "echo OK"
ssh m2 "echo OK"
ssh m3 "echo OK"
ssh w1 "echo OK"
ssh w2 "echo OK"
ssh hb "echo OK"
ssh us "echo OK"
```
